# Expense Calculator

Простой консольный калькулятор расходов.  
Учебный пет-проект для отработки базовых навыков программирования и работы с данными.

---

## Зачем этот проект

Проект сделан в учебных целях, чтобы:
- научиться работать с пользовательским вводом
- хранить и обрабатывать данные в памяти
- считать суммы и делать простую аналитику
- привыкнуть к базовой структуре приложения

---

## Функциональность

- Добавление расхода:
  - категория
  - сумма
  - дата
- Просмотр общей суммы расходов:
  - за выбранный день
  - за выбранный месяц
- Фильтрация расходов по категории

---

## Как это работает

Программа запускается в консоли и предлагает меню с доступными действиями.  
Все расходы хранятся в памяти (в списке) и теряются после выхода из программы.

---

## Структура проекта

Примерная структура:

- `Expense` — класс, описывающий один расход  
- `ExpenseService` — логика работы с расходами (добавление, фильтрация, подсчёты)  
- `Main` / `App` — точка входа, меню и взаимодействие с пользователем  

---

## Валидация данных

Реализованы базовые проверки:
- сумма не может быть отрицательной
- категория не может быть пустой
- дата должна быть введена в корректном формате
- обработка неверного пользовательского ввода

---
